## Тавтай морилно уу

Энэ хэсэгт Номин Даатгалын API-г тайлбарласан болно. Энэхүү API нь эрх бүхий хэрэглэгчдэд даатгалын гэрээний мэдээллийг хайх боломжийг олгодог.

## Баталгаажуулалт (Authentication)

Хандалт нь **Bearer Token** ашиглан хийгдэнэ. Та эхлээд **Хэрэглэгч нэвтрэх** (Authenticate) endpoint-ийг ашиглан өөрийн `username`, `password`, болон `db` мэдээллийг илгээж токен авах шаардлагатай.

Токен авсны дараа дараагийн бүх хүсэлтийн `Authorization` хэсэгт уг токеноо дараах байдлаар оруулна:

```http
Authorization: Bearer <your_token>
```
