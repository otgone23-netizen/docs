## Техник хангамж

Хөгжүүлэлт хийхэд хангалттай үзүүлэлттэй компьютер (Mac/Linux санал болгож байна).

## Программ хангамжууд

### 1. Git
```bash
brew install git
```

### 2. VS Code / JetBrains IDE
Санал болгох өргөтгөлүүд:
*   Python
*   Pylance
*   Rainbow CSV
*   Dracula Theme Official



### 3. PostgreSQL
```bash
brew install postgresql@16
```

### 4. Python & Virtualenv
Odoo хөгжүүлэлтэд зориулан Python 3.10+ ашиглах.

## Дотоод эрхүүд

*   **GitLab:** Баримт бичиг болон кодын агуулах.
*   **Viber:** Багийн дотоод харилцаа.
