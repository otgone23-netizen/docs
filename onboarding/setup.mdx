## Техник хангамж

Таны ажлын компьютерын үзүүлэлт хөгжүүлэлт хийхэд хангалттай байх шаардлагатай. Хэрэв асуудал гарвал IT-ийн хэлтэст хандаарай. 

## Шаардлагатай программ хангамжууд

Хөгжүүлэгчийн хувьд дараах багажуудыг суулгасан байх шаардлагатай:

### 1. Git
Хувилбарын удирдлагын систем.
```bash
brew install git
```

### 2. Docker & Docker Compose
Системийг контейнержүүлсэн орчинд ажиллуулах.
*   [Docker Desktop](https://www.docker.com/products/docker-desktop/) суулгах.

### 3. VS Code / JetBrains IDE
Өөрт таалагдах код редакторыг сонгон суулгах. Санал болгох өргөтгөлүүд:
*   Python
*   Odoo Snippets
*   Flutter / Dart (Хэрэв апп хөгжүүлэлт хийх бол)

### 4. PostgreSQL
Өгөгдлийн сантай ажиллах.
```bash
brew install postgresql@15
```

## Дотоод системүүд

Дараах системүүдэд эрх нээлгэх:
*   **GitHub/Bitbucket:** Кодын агуулах.
*   **Odoo:** Компанийн ERP систем.
*   **Slack/Discord:** Дотоод харилцаа.
